Detailed Description

Costanti di calibrazione hardware per i motori.

Author
Gruppo 2

Questo file contiene i valori empirici rilevati per centrare il segnale PWM (punto neutro) e definire l'escursione massima (scaling) per ogni motore. Questi valori compensano le differenze fisiche tra gli ESC/Driver.
