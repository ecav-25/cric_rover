Detailed Description

Implementazione del protocollo di comunicazione con il controller.

Author
Gruppo 2

Questo file gestisce la comunicazione I2C master verso il controller remoto, validando i pacchetti ricevuti tramite checksum CRC-8 e formattando i pacchetti di telemetria in uscita.

◆ telecontrol_read()

ControllerStatus_t telecontrol_read ( Controller_t * telecontrol)

Legge il pacchetto dati dal controller via I2C e verifica il CRC.

Legge lo stato attuale degli input dal controller remoto.

La funzione esegue una lettura bloccante (con timeout a 1ms). Se la lettura I2C ha successo, calcola il CRC sui campi ricevuti (Data + Percentage + Alive) e lo confronta con il campo crc_value ricevuto.

Parameters
telecontrolPuntatore all'handle del controller.
Returns
ControllerStatus_t CONTROLLER_OK, CONTROLLER_ERR_CRC o CONTROLLER_ERR_COMM.
