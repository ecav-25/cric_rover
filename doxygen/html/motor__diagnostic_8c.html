Detailed Description

Implementazione del modulo di diagnostica motori.

Author
Gruppo 2

Questo file gestisce l'acquisizione dei dati in buffer circolari (Ping-Pong) e il calcolo dell'errore IAE (Integral Absolute Error) per rilevare anomalie nel comportamento dei motori DC.

MotorDiag_Status_t motor_diag_process ( MotorDiag_Handle_t * hdiag)

Elabora i dati raccolti per determinare lo stato di salute del motore.

Analizza il buffer "pieno" (quello appena completato, opposto a quello attivo). Calcola l'errore IAE e aggiorna lo stato di salute.
