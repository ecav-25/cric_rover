Detailed Description

Unit Tests per il modulo di diagnostica avanzata dei motori.

Author
Gruppo 2

Questa suite di test valida l'algoritmo di monitoraggio della salute dei motori. Il modulo utilizza una tecnica di analisi dell'errore integrale (IAE) con compensazione del ritardo di fase (delay shift).
