Detailed Description

Unit Tests per il modulo encoder.

Author
Gruppo 2

I test verificano l'inizializzazione del timer in modalità encoder, la gestione del calcolo RPM in senso orario e antiorario, e la corretta gestione degli overflow del contatore (ARR).

void test_encoder_readRPM_forward_counting ( void )
Test
Calcolo RPM standard in avanti (conteggio crescente).

Formula: (60 * diff) / (PPR * Period * Gear) Con PPR=100, Period=0.1s, Gear=1.0, Diff=10 -> RPM = (600 / 10) = 60 RPM. Nota: Il driver inverte il segno per il conteggio crescente.
