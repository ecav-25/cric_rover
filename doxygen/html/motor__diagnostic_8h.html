Detailed Description

Modulo di diagnostica per il monitoraggio della salute dei motori.

Author
Gruppo 2
Copyright
Copyright (c) 2026 Gruppo 2. Rilasciato sotto licenza GPLv3 (consultare il file LICENSE per i dettagli).

Il modulo confronta il setpoint (riferimento) con la velocità misurata (encoder) utilizzando un buffer circolare. Calcola l'errore cumulativo (area) per rilevare discrepanze che indicano malfunzionamenti o guasti hardware.
