Detailed Description

Implementazione del Hardware Abstraction Layer (HAL) per Board 1.

Author
Gruppo 2
  • Include l'implementazione delle funzioni di comunicazione UART via DMA, calcolo CRC-32 hardware, gestione GPIO e primitive di temporizzazione. Il driver è progettato per operare in un ambiente multitasking FreeRTOS.
