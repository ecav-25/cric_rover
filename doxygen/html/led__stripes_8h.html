Detailed Description

Driver per la gestione delle strisce LED (fanaleria e targa).

Author
Gruppo 2

Questo modulo implementa il controllo della striscia LED utilizzando periferiche PWM e DMA. Gestisce sia la fanaleria posteriore dinamica che la retroilluminazione della targa del rover.

◆ led_render()

led_status_t led_render ( void )

Avvia la trasmissione dei dati via DMA per aggiornare i LED fisici.

Applica le modifiche ai colori dei LED della fanaleria e della targa.

Returns
LED_STRIPE_OK se la trasmissione è avviata, LED_BUSY se il DMA è già attivo.

Avvia la trasmissione dei dati via DMA per aggiornare i LED fisici.

Utilizza un meccanismo di buffer circolare per convertire i dati RGB in segnali PWM compatibili con WS2812/SK6812.

Returns
led_status_t LED_STRIPE_OK.
