Test
Verifica la conversione a 25.0°C.

Calcolo atteso: V_sense = 760mV (Parametro V25). Raw = (760 * 4095) / 3300 = 943.45 -> 943.

◆ test_temp_get_celsius_at_high_temp()

void test_temp_get_celsius_at_high_temp ( void )
Test
Verifica la conversione a 45.0°C.

Delta T = 20°C. Delta V = 20 * 2.5mV = 50mV. V_sense = 760 + 50 = 810mV. Raw = (810 * 4095) / 3300 = 1005.
