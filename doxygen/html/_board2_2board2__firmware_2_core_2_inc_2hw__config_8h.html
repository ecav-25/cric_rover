Detailed Description

Definizioni e tipi per la configurazione hardware astratta.

Author
Gruppo 2

Questo file espone le strutture dati e gli enumeratori utilizzati per mappare le risorse hardware (Timer, I2C, GPIO) ai driver logici del sistema (Motori, Sensori, Controller). Le istanze reali sono definite in hw_config.c.
