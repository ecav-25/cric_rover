Detailed Description

Hardware configuration definition for Board 2 (Master).

Author
Gruppo 2

This file implements the hardware abstraction layer configuration, mapping physical resources (Timers, I2C, GPIOs) to logical driver instances (Motors, IMU, Ultrasonic Sensors, Controller).
