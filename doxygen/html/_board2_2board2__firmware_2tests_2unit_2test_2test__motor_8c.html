Detailed Description

Unit Tests per il driver motori DC (PWM).

Author
Gruppo 2

Questa suite valida la logica di calibrazione hardware. Utilizza un'istanza reale della struttura TIM_TypeDef (definita in hal.h) per intercettare le scritture nei registri CCR effettuate dal driver.
